## RESTful 控制器
| 注解 | HTTP方法 | 使用场景 |
| --- | --- | --- |
| @GetMapping | HTTP GET 请求 | 读取资源数据 |
| @PostMapping | HTTP POST 请求 | 创建资源 |
| @PutMapping | HTTP PUT 请求 | 更新资源 |
| @PatchMapping | HTTP PATCH 请求 | 更新资源 | 
| @DeleteMapping | HTTP DELETE 请求| 删除资源 | 
| @RequestMapping | | 通用请求处理 |
