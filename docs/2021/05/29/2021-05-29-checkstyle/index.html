checkstyle 使用

在gradle中配置checkstyle

checkstyle.xml

默认路径 /config/checkstyle/checkstyle.xml

google 的checkstyle.xml，做了一些更改，尽量与IDEA保持一致，减少配置成本

主要更改

  • 每行最大字符数（与IDEA保持一致）
  • 缩进，4个字符（换行策略与IDEA保持一致）
  • 不检查连续大写字母的长度，AABBcc是可被接受的
  • import顺序，第三方 -> javax|java -> static（与IDEA保持一致）
  • 暂不考虑Javadoc
