## node安装
在官方下载网站下载 pkg 安装包，直接点击安装即可。https://nodejs.org/en/download/

#### 查看当前node版本
```shell
node -v 
```

## npm安装
npm在Node.js安装的时候顺带装好了

#### 查看npm版本
```shell
npm -v
```

## node版本管理工具n
#### 安装
```shell
sudo npm install -g n
```


#### 升级node到最新稳定版
```shell
sudo n stable
```

#### 升级node到指定版本
```shell
sudo n v10
```

#### 切换当前使用的node版本
```shell
n
```
选择指定版本
